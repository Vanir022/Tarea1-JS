Práctica con 3 eventos y 3 ejercicios con estructuras de control

Eventos de JavaScript

Evento 1: onchange

Selecciona un color para cambiar el fondo del cuadro:

Código:
onchange="cambiarColorFondo()" - Se ejecuta cuando cambia el valor del select

Evento 2: onsubmit

Completa el formulario y envíalo (se validará antes del envío):





Completa el formulario y haz clic en enviar
Código:
onsubmit="validarFormulario(event)" - Se ejecuta al enviar el formulario

Evento 3: onresize

Cambia el tamaño de la ventana del navegador para ver la información actualizada:

Ancho: 0px | Alto: 0px
Código:
window.onresize = actualizarTamano - Se ejecuta al cambiar el tamaño de ventana
